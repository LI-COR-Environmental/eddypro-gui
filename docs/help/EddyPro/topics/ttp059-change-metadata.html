Mistakes happen. During the busy hustle of an eddy covariance installation, you might accidentally enter some metadata values incorrectly in the LI-7500RS/LI-7500DS/LI-7200RS Windows Interface Software.   2. Enter the Project Name, and select LI-COR GHG.
