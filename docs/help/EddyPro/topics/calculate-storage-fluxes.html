Storage fluxes are calculated in EddyPro following the approach outlined in Aubinet et al., (2001). Mass storage fluxes associated with gases are calculated as:

In practice, integrals are approximated using the trapezium rule and time derivatives are approximated by the discrete difference of mean concentrations measured during consecutive time averaging intervals.

In eddy covariance applications where profile concentration data are not available, storage fluxes are approximated by using one-point time derivatives, assuming that all gradients nullify at ground level and that the profile is linear from the measurement point to the ground. Clearly, this is an extreme assumption that probably is not true in most situations. Therefore, we encourage you to consider storage fluxes provided by EddyPro as purely indicative. For the same reason, storage fluxes are not summed to turbulent fluxes to provide Net Ecosystem Exchange (NEE) estimations, which would be affected by unacceptable inaccuracy.
