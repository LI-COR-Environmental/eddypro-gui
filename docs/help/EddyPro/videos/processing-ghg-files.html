Video: Processing .ghg Files

 

This four-minute video shows how to process .ghg files in Express Mode. .ghg files are logged by the LI‑7500RS and LI‑7200RS gas analyzers and provide the simplest way to compute fluxes in EddyPro. Download a sample data set and follow along.
